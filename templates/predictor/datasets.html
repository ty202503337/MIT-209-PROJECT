Raw Data Baseline

This dataset underwent basic cleaning (handling missing values, encoding categorical variables like Education and Marital Status). However, the numerical values were left exactly as they were in the real world. It serves as our baseline control group.

Scaled Data

Machine learning models can be biased by massive differences in number size (e.g., comparing an 'Income' of 80,000 to an 'Age' of 40). We applied Standardization here to scale all features to have a mean of zero and a variance of one, leveling the mathematical playing field.

PCA Data

Using Principal Component Analysis (PCA), we applied dimensionality reduction. This strategy attempts to compress the dataset by removing noise and combining highly correlated features into principal components, keeping only the most important statistical variance.
